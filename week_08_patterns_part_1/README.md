# Week 8: Interview Patterns Part 1

## The Big Shift

This is where you stop learning SQL topics one by one and start thinking in reusable interview patterns.

## Patterns This Week

- top-N per group
- running totals
- month-over-month and year-over-year growth
- medians and percentiles
- pivot and unpivot

## Pattern Recognition Questions

When you read a problem, ask:

1. Is this ranking inside groups?
2. Is this comparison against previous time periods?
3. Does the output keep every row or collapse rows?
4. Are ties important?

## Interview Lens

This week is where hard problems start looking repetitive in a good way. Different story, same pattern.
