# Funnel and A/B Test

## Use When

You need conversion between ordered stages or metric comparison across experiment groups.

## Core Trick

For funnels:

1. define stage membership
2. deduplicate users at each stage
3. compare counts stage by stage

For A/B tests:

1. define experiment exposure
2. define outcome
3. aggregate per variant

## Worked Ideas

- page view to cart to purchase
- signup to activation
- treatment vs control conversion

## Interview Note

Always clarify whether stages must happen in order and within the same session.
