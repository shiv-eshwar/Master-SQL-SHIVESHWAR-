# Median and Percentiles

## Use When

Average is too sensitive to skew and you need a robust distribution summary.

## Core Tools

- `PERCENTILE_CONT`
- `PERCENTILE_DISC`
- window-based ranking approaches

## Worked Ideas

- median ride fare
- 90th percentile latency
- quartiles of order value

## Interview Note

Be able to explain the difference between continuous and discrete percentile logic.
