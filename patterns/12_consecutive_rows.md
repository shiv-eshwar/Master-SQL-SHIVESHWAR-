# Consecutive Rows

## Use When

You need previous-next comparisons, streak starts, or change detection across ordered records.

## Core Tools

- `LAG`
- `LEAD`
- `ROW_NUMBER`
- cumulative `SUM`

## Worked Ideas

- price change detection
- first order after churn
- consecutive login days

## Interview Note

This pattern often overlaps with gaps and islands, but here the emphasis is on row-to-row transitions.
