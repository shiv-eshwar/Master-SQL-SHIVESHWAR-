# Pivot and Unpivot

## Use When

You need categories turned into columns or columns turned back into rows.

## Core Trick

For interview SQL, manual pivoting with conditional aggregation is usually enough.

## Worked Ideas

- status counts by payment method
- monthly metrics by channel
- converting multiple metric columns back into a tall table

## Interview Note

Manual pivoting is often more portable and easier to explain than dialect-specific helpers.
