# Week 12: Interview Mastery and Timed Drills

## Final Goal

The goal this week is not to learn more syntax. It is to make your thinking fast, calm, and repeatable.

## Your Interview Loop

1. Clarify the requested output.
2. State the output grain.
3. Identify the pattern.
4. Sketch CTE layers.
5. Write the query.
6. Test edge cases mentally.
7. Offer a performance note.

## Typical Follow-Ups

- Do it without a window function.
- Do it with a window function.
- How would you index it?
- What happens with ties?
- What if there are duplicate events?
- What if the data volume is 1000x larger?

## Final Advice

Practice saying your thought process aloud. Senior interview performance is partly technical and partly communication under pressure.
