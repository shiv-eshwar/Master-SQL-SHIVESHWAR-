# Regex and Strings Cheatsheet

- wildcard matching: `LIKE`, `ILIKE`
- regex match: `~`
- case-insensitive regex: `~*`
- uppercase/lowercase: `UPPER`, `LOWER`
- cleanup: `TRIM`, `REPLACE`, `REGEXP_REPLACE`
- split patterns: `SPLIT_PART`

Use regex when the pattern is structural, not when a simple prefix/suffix search is enough.
